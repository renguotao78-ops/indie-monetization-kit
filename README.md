## 核心工作流示例

### 1. monetization-release.yml
打版本 tag 后自动打包产品、创建 GitHub Release、发送客户通知。

### 2. content-automation.yml
自动生成并推送营销内容到 X / Reddit / 小红书等。

### 3. customer-onboarding.yml
新订单自动欢迎并交付产品。
